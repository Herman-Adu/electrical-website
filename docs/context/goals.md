# Project Goals

**Primary goal:** Convert website visitors into enquiries/bookings for electrical services

**Secondary goals:**
1. Showcase expertise and portfolio of completed projects
2. Build trust through professional design and clear information
3. Rank locally for electrical services search terms (SEO)
4. Streamline quote/contact process to reduce friction

**Success metrics:**
- Contact form submissions
- Quote request completions
- Time-on-page for services and projects sections
- Mobile experience quality (primary device for local searches)

**Content strategy:**
- Service pages that answer "can you do X?" clearly
- Project portfolio showing real completed work
- Clear pricing/process transparency where possible
- Local area signals for SEO
